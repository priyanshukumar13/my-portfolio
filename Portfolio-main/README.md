## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/portfolio.git
```

Go to the project folder:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
